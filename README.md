# Cricket.com
see match schedule

## **Installation Requirements**
### Fork the repo, then clone it
### install all dependencies, of client and server with 'npm install'
### start the server 'npm run dev'
### start the client 'npm start'

## **Tech Stack Used**

### MongoDB

### Chackra UI
